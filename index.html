CARA

CARA (Career Recommendation and Assistance) is a web-based career guidance application that helps users identify suitable IT career paths based on their skills, interests, and personality traits. The system uses a machine learning model trained on a dataset containing 36 user attributes to predict 2–3 potential job roles for each individual. In addition to personalized recommendations, CARA provides career roadmaps, job search functionality, and a recruiter portal. I was responsible for building and integrating the ML model, which gave me hands-on experience in data preprocessing, model selection, and real-world application of AI in career planning.
